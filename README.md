# SimpleAutoFramer C++ 

## Installation

### Prerequisites

- OpenCV
- V4l2loopback
- CMake
- Make
- Apindicator3
- Gtk3
- ...

Check the [install.sh](install.sh) script for more details.  

### Instructions

Tested on Ubuntu 24.04 Gnome only.

## Install

```bash
git clone https://github.com/Rimsoo/SimpleAutoFramer.git
cd SimpleAutoFrame
chmod +x install.sh
sudo ./install.sh
simpleautoframe
```

## Usage

```bash
simpleautoframe
```

## Unsintall

```bash
sudo simpleautoframe-uninstall
```
